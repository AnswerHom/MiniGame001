## 概述

### v6 核心目标

本次迭代聚焦于**增强美术效果**和**角色表现力**，让游戏从功能性向沉浸感迈进。主要目标：

1. **场景沉浸感** - 打造真实生动的便利店场景
2. **角色表现力** - 小人具备丰富的动作和表情
3. **视觉丰富度** - 粒子效果、光效、音效提示
4. **交互反馈** - 流畅的动画和及时的玩家反馈

### 与 v5 的区别

| 功能 | v5 | v6 |
|------|-----|-----|
| 场景 | 静态背景 | 动态可交互场景 |
| 小人 | 静态/简单移动 | 完整移动系统+多动作 |
| 美术 | 基础UI | 粒子+光效+动画 |
| 反馈 | 基础提示 | 沉浸式反馈 |

```javascript
    LEAVING: 'leaving',         // 离开中

```

#### 移动控制逻辑

```javascript
class CustomerMovement {
    // 路径导航
    navigateTo(targetPoint, speed) {
        const direction = this.calculateDirection(targetPoint);
        this.setAnimation('walking');
        this.move(direction.x * speed, direction.y * speed);
    }

    // 路径平滑处理
    smoothPath(points) {
        // 使用贝塞尔曲线平滑路径
        return this.bezierCurve(points, 0.5);
    }

    // 到达检测
    checkArrival(targetPoint, threshold = 10) {
        const distance = this.getDistance(this.position, targetPoint);
        return distance < threshold;
    }
}
```

## 动作系统

### 1. 走路动作 (Walk)

#### 动画序列

```
帧1: 站立 → 帧2: 左脚抬起 → 帧3: 左脚迈出 → 帧4: 身体前倾
      → 帧5: 右脚抬起 → 帧6: 右脚迈出 → 帧7: 身体前倾 → 循环
```

#### 动作参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 动画帧数 | 8帧 | 完整行走周期 |
| 循环播放 | 是 | 持续行走时循环 |
| 播放速度 | 8 FPS | 每秒8帧 |
| 手臂摆动 | 15° | 摆动角度 |

#### 视觉效果
- 脚步轻微弹跳（3px）
- 身体左右轻微摇摆（2px）
- 影子跟随

#### 动作分解

| 阶段 | 动作 | 动画 | 音效 |
|------|------|------|------|
| 1 | 拿起商品 | 手部上抬 | 拿取声 |
| 2 | 查看商品 | 转身+低头 | - |
| 3 | 放入购物车 | 手部前伸 | 放入声 |
