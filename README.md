# 便利店小店主 v2

## 🎮 游戏介绍

《便利店小店主》是一款模拟经营游戏，玩家经营一家便利店，通过调整价格、进货、服务顾客来赚取金币，升级设施，成为最成功的店主！

## ✨ v2 新功能

### 1. 🎨 场景系统
- **完整的便利店场景布局**：包含货架区、收银台区、排队区
- **动态背景**：天空、建筑、装饰物
- **视觉反馈**：商品展示、顾客移动

### 2. 👥 顾客排队系统
- **可视化排队**：顾客在排队区排队等待服务
- **优先级系统**：VIP顾客优先服务
- **行为逻辑**：
  - 等待中显示倒计时
  - 价格合适时购买
  - 价格不合适或等待过久时离开
- **排队管理**：最多支持8人排队

### 3. 🎯 表现优化
- **美观的UI设计**：渐变色、阴影、圆角
- **流畅的动画**：顾客移动、商品展示、升级特效
- **实时反馈**：购买成功、升级提示、排队变化
- **响应式布局**：适配不同屏幕尺寸

## 🕹️ 游戏说明

### 基本操作
- **调整价格**：点击商品的价格 +/-
- **进货**：点击进货按钮补充库存
- **升级设施**：点击升级按钮提升收银台和货架

### 游戏策略
1. **定价策略**：价格太低利润少，价格太高顾客不买
2. **库存管理**：及时进货，避免缺货
3. **设施升级**：升级收银台提升服务速度，升级货架增加库存上限
4. **顾客服务**：VIP顾客优先服务，避免顾客等待过久

### 顾客类型
| 顾客类型 | 图标 | 消费范围 | 等待时间 | 优先级 |
|---------|------|---------|---------|--------|
| 普通顾客 | 😊 | $5-20 | 5秒 | 1 |
| 急切顾客 | 😰 | $10-30 | 3秒 | 2 |
| VIP顾客 | 👑 | $20-50 | 8秒 | 3 |

## 🏗️ 游戏系统

### 商品系统
- **饮料**：🥤 - 热销商品，利润率 30%
- **零食**：🍪 - 热销商品，利润率 25%
- **日用品**：🧴 - 稳定商品，利润率 20%

### 设施升级
- **收银台**：提升服务速度
  - Lv1: 基础
  - Lv2: 提升 10% 速度
  - Lv3: 提升 20% 速度
  - Lv4: 提升 30% 速度
  - Lv5: 提升 40% 速度

- **货架**：增加库存上限
  - Lv1: 基础
  - Lv2: +5 库存
  - Lv3: +10 库存
  - Lv4: +15 库存
  - Lv5: +20 库存

## 🚀 如何运行

### 方式1：直接打开
1. 下载 `便利店小店主.html` 文件
2. 双击打开或在浏览器中打开

### 方式2：本地服务器
```bash
# 使用 Python
python -m http.server 8000
# 访问 http://localhost:8000

# 使用 Node.js
npx serve
# 访问 http://localhost:3000
```

### 方式3：Netlify Drop
1. 访问 https://app.netlify.com/drop
2. 拖拽 `便利店小店主.html` 到网页
3. 立即获得访问链接

## 📊 游戏数据

### 初始状态
- 金币：$100
- 等级：1
- 库存：每种商品 10 个

### 升级成本
- 收银台升级：$500, $1000, $2000, $5000
- 货架升级：$300, $600, $1200, $3000

### 顾客生成
- 生成频率：每 5 秒
- 最大排队：8 人
- 排队消失：等待超过 8 秒
